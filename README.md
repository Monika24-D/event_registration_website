# Inter College Technical Fest – Event Registration Website

This project is a simple Event Registration Website developed using HTML.  
It represents an Inter College Technical Fest and provides information about technical events, non-technical events, and workshops.

---

## Live Demo

Live Website (GitHub Pages):  
https://monika24-d.github.io/event_registration_website/

GitHub Repository:  
https://github.com/Monika24-D/event_registration_website

---

## Features

- Homepage for Inter College Technical Fest
- Technical and non-technical event details
- Workshop information
- Event list displayed using tables
- Separate registration page
- Image integration
- Beginner-friendly HTML structure

---

## Technologies Used

- HTML5
- Basic inline styling
- GitHub Pages for deployment
